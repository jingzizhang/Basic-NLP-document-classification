# Basic-NLP-document-classification

  classify documents from eight groups 
  ('comp.graphics',
  'comp.os.ms-windows.misc',
  'comp.sys.ibm.pc.hardware',
  'comp.sys.mac.hardware',
  'rec.autos',
  'rec.motorcycles',
  'rec.sport.baseball',
  'rec.sport.hockey')
  into two groups (comp and rec) by SVM and Naive Bayesian.
